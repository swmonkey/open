# Markdown Key Elements

This file demonstrates the most common and useful **Markdown (.md)** elements. You can store this file in **GitHub** or on a **web server** and render it anywhere without duplication.

---

## Headings

# H1 – Main Title

## H2 – Section

### H3 – Subsection

#### H4 – Minor heading

* **Bold text**
* *Italic text*

* ~~Strikethrough~~
* `Inline code`
